It is a project on Apriori algorithm during "Introduction to Algorithm" class in Masters course.

In this project, there are a few .csv (Comma Separated Vectors) files with inputs of 1000, 5000, 20000, and 75000 data respectively.
Also, providing minimum support between 1 and 1000 was another requirement/input.

Implemented the logic and then presentend this project on webpage using Google Cloud Platform.
